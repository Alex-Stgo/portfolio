Web scraping , web harvesting, or web data extraction is data scraping used for extracting data from websites. Web scraping is the process of collecting structured web data in an automated process. Some of the main use cases of web scraping include price monitoring, price intelligence, news monitoring, str lead generation, and market research among many others.

Details

The web scraping is focused on retrieve information about books in stock through the information listed in the page Books to scrape.
The search will be performed thorugh the title of the book. The query will return the i elements more close with the title typed given a cutoff. The similarities between strings are performed by the difflib which is based on the algorithm published in the late 1980’s by Ratcliff and Obershelp under the hyperbolic name “gestalt pattern matching" .
